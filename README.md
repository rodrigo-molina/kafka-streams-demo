# Kafka Streams Example

## Run Kafka from docker compose
```
$ cd  .\src\test\resources
$ docker-compose up
```

## Run App with gradle
```
$ gradle run
```


## Run App java -jar
```
$ gradle jar
$ java -jar build/libs/kafka-streams-demo-0.0.1.jar
```


## Run Docker container
```
$ gradle startContainer
```
